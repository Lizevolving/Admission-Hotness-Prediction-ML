# 数据字典 (Data Dictionary)

## 数据集概述
本数据集包含高校各专业的招生数据，用于分析和预测专业报考热度。

## 字段定义

| 字段名 | 数据类型 | 描述 | 约束/说明 | 示例值 |
|--------|----------|------|-----------|--------|
| **year** | integer | 招生年份 | 2019-2023 | 2023 |
| **province** | string | 招生省份 | 中国各省级行政区 | "北京", "上海" |
| **school_name** | string | 高校名称 | 中国高校名称 | "清华大学", "北京大学" |
| **school_tier** | string | 高校层次 | 分类: "985", "211", "普通本科", "专科" | "985" |
| **major_name** | string | 专业名称 | 高校专业名称 | "计算机科学与技术" |
| **category** | string | 考生科类 | 分类: "理科", "文科", "综合", "工科" | "理科" |
| **plan_quota** | integer | 计划招生名额 | 正整数, 1-5000 | 120 |
| **apply_num** | integer | 实际报考人数 | 正整数, 1-10000 | 850 |
| **min_score** | integer | 最低录取分数 | 正整数, 0-750 | 650 |
| **min_score_rank** | integer | 最低录取分数省内排名 | 正整数, 1-100000 | 1500 |
| **hotness_index** | float | 报考热度指数 | 目标变量, apply_num/plan_quota | 7.08 |

## 字段详细说明

### 核心特征字段

#### `year` (年份)
- **用途**: 分析时间趋势，控制年份效应
- **范围**: 2019-2023
- **重要性**: 中等

#### `province` (省份)
- **用途**: 反映地域差异，教育资源分布
- **编码方式**: One-hot编码
- **重要性**: 高

#### `school_tier` (高校层次)
- **用途**: 反映学校声誉和竞争力
- **取值说明**:
  - "985": 985工程大学
  - "211": 211工程大学  
  - "普通本科": 普通本科院校
  - "专科": 专科院校
- **编码方式**: One-hot编码
- **重要性**: 高

#### `category` (科类)
- **用途**: 反映专业类别差异
- **取值说明**:
  - "理科": 理科类专业
  - "文科": 文科类专业
  - "综合": 综合类专业
  - "工科": 工科类专业
- **编码方式**: One-hot编码
- **重要性**: 中等

### 数值特征字段

#### `plan_quota` (计划招生名额)
- **用途**: 反映学校招生规模
- **数据分布**: 偏态分布，可能需要log变换
- **重要性**: 高

#### `apply_num` (实际报考人数)
- **用途**: 直接反映报考热度
- **数据分布**: 偏态分布，可能需要log变换
- **重要性**: 高

#### `min_score` (最低录取分数)
- **用途**: 反映专业竞争激烈程度
- **数据分布**: 近似正态分布
- **重要性**: 高

#### `min_score_rank` (省内排名)
- **用途**: 比分数更稳定的竞争指标
- **数据分布**: 偏态分布，可能需要log变换
- **重要性**: 很高

### 目标变量

#### `hotness_index` (报考热度指数)
- **计算公式**: `hotness_index = apply_num / plan_quota`
- **含义**: 报录比，反映竞争激烈程度
- **数据分布**: 右偏分布，范围通常在0-20之间
- **特点**: 
  - >10: 极热门专业
  - 5-10: 热门专业
  - 2-5: 中等热门
  - <2: 冷门专业

## 数据质量要求

### 完整性
- 所有字段不允许为空值
- 关键字段(year, school_name, major_name)必须完整

### 一致性
- 同一学校在不同年份的school_tier应保持一致
- province应使用标准省份名称

### 合理性
- apply_num >= plan_quota (报考人数不少于计划名额)
- min_score应在合理范围内(0-750)
- min_score_rank应为正整数

## 特征工程建议

### 数值特征
1. **log变换**: 对plan_quota, apply_num, min_score_rank进行log变换
2. **标准化**: 对数值特征进行标准化处理

### 分类特征
1. **One-hot编码**: 对province, school_tier, category进行编码
2. **特征合并**: 可考虑将year与category组合特征

### 交互特征
1. **组合特征**: plan_quota * school_tier 反映不同层次学校的招生规模效应
2. **比率特征**: min_score / min_score_rank 反映分数效率

## 数据生成规则

为确保数据真实性，模拟数据应遵循以下规则：
1. 985院校的热度指数普遍高于普通院校
2. 计算机、金融等专业热度高于传统专业
3. 经济发达省份的专业热度普遍较高
4. 热度分布呈现右偏特征
